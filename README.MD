# 基于 Vue3 + SpringBoot 的在线教育直播平台实现方案
 
**架构：** 后端 Spring Boot 3 + MyBatis-Plus　前端 Vue3 + Vite　数据库 MySQL 8
 
**环境：** JDK 17　Node.js 22　MySQL 8（UTF-8mb4 编码）　文件存储基于本地项目目录
 
**直播方案：** 集成声网 Agora Web SDK（摄像头推流 / 屏幕共享 / 连麦）+ SpringBoot WebSocket（聊天 / 弹幕 / 投票信令）
 
**权限：** 三角色（管理员、教师、学生）
 
---
 
## 一、管理员功能
 
### 管理总览
- 登录、退出、修改密码
- 数据概览仪表盘（注册用户总数、课程总数、今日直播场次、平台活跃人数统计图表）
### 用户管理
- 学生账号管理（查看学生列表、搜索、禁用/启用账号、重置密码）
- 教师账号管理（新增教师账号、编辑信息、禁用/启用、删除）
- 管理员账号管理（新增子管理员、分配角色）
### 课程管理
- 查看全平台课程列表（支持按分类、教师、状态筛选）
- 课程审核（教师发布课程后需管理员审核上架）
- 课程下架 / 删除
### 分类管理
- 课程分类新增、编辑、删除（如编程开发、设计创意、英语口语等）
- 支持一级分类配置
### 直播管理
- 查看全平台直播间列表及状态（待开始 / 直播中 / 已结束）
- 强制关闭违规直播间
- 直播历史记录查看
### 弹幕管理
- 查看全平台弹幕记录（支持按直播间、关键词、时间筛选）
- 删除违规弹幕
- 关键词屏蔽配置（设置屏蔽词，发送时后端过滤）
---
 
## 二、教师功能
 
### 个人信息
- 登录、退出、修改密码
- 编辑个人资料（头像、简介、教龄、擅长方向）
- 头像上传（存储至本地 `/uploads/avatar/` 目录）
### 课程管理
- 课程列表（我发布的全部课程，展示状态：草稿 / 审核中 / 已上架 / 已下架）
- 新建课程（填写课程名称、封面图、简介、分类，价格设为 0 即免费）
- 课程章节管理（新增章节、上传章节视频文件至本地 `/uploads/video/` 目录）
- 编辑 / 删除草稿状态的课程
- 提交审核（提交给管理员审核上架）
### 直播管理
- 创建直播间（填写直播标题、封面、简介、预定开始时间）
- 开始直播（调用 Agora SDK 开启摄像头或屏幕共享推流）
- 直播控制台（切换摄像头/屏幕共享、开启/关闭麦克风、查看在线人数、查看实时聊天）
- 发起投票（设置投票题目和选项，实时推送给学生，查看投票结果）
- 申请连麦管理（查看学生连麦申请列表，同意/拒绝，与学生进行音视频连麦）
- 结束直播（直播结束后自动生成回放记录条目）
- 直播历史列表（查看历史直播间列表）
### 回放管理
- 回放列表（查看我的历史直播对应的回放记录）
- 回放标题 / 简介编辑
- 回放上传（支持手动上传录制好的视频文件至本地 `/uploads/replay/` 目录）
- 删除回放
### 作业管理
- 发布作业（选择关联课程、填写作业标题、作业描述、截止时间）
- 作业列表（查看已发布的全部作业）
- 批改作业（查看学生提交的作业内容/附件，填写评语，录入分数）
- 作业统计（查看某作业的提交人数 / 未提交人数）

## 三、学生功能
 
### 登录注册
- 注册（用户名、密码、邮箱）
- 登录、退出
- 修改密码、编辑个人资料、上传头像
### 首页
- 轮播图（由管理员配置）
- 推荐课程（取收藏数量最多的前 8 门）
- 正在直播（展示当前状态为直播中的直播间，实时更新）
- 近期直播预告（展示待开始的直播间列表）
### 课程中心
- 课程列表（支持按分类筛选、关键词搜索、排序）
- 课程详情页（封面、简介、章节目录、教师信息）
- 课程章节视频播放（本地视频通过后端流式传输至前端播放器）
- 收藏课程 / 取消收藏
### 直播中心
- 直播间列表（展示全部直播间：直播中 / 待开始 / 已结束）
- 进入直播间（调用 Agora SDK 拉流观看直播画面）
- 实时聊天（WebSocket，所有观众可发消息）
- 发送弹幕（WebSocket 广播，前端 Canvas 渲染滚动弹幕）
- 参与投票（教师发起投票后弹出投票面板，提交选项，查看实时结果）
- 申请连麦（发送连麦申请，等待教师同意后开启本地麦克风/摄像头接入）
- 在线人数显示
### 课程回放
- 回放列表（展示全平台已发布的直播回放）
- 回放详情页（视频播放 + 回放简介）
### 我的学习
- 学习记录列表（我观看过的课程，显示最后观看章节）
- 继续学习（跳转至上次观看的章节）
- 学习进度（已完成章节数 / 总章节数）
### 我的收藏
- 已收藏课程列表（分页展示）
- 取消收藏
### 我的作业
- 作业列表（我需要完成的全部作业，显示截止时间、状态：未提交 / 已提交 / 已批改）
- 提交作业（填写作业内容文本，可上传附件至本地 `/uploads/homework/` 目录）
- 查看批改结果（教师评语 + 分数）
