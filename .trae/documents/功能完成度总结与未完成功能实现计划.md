# 功能完成度总结与分会话实现计划

## 依据与范围

- 需求依据：项目根目录 `README.MD` 中的管理员、教师、学生功能清单。
- 系统依据：当前代码中的后端 Controller/Service/Entity/SQL、前端路由/API/页面实现。
- 完成判定：以“前后端均有对应实现、接口可被页面调用、权限边界基本明确”为主要判定标准；计划阶段不执行运行测试，最终“测试无误可用”以每次实现后的 lint、typecheck、build、后端编译和角色流程验收为准。

## 当前系统已完成且具备可用链路的功能

### 基础架构与公共能力

1. 后端 Spring Boot 3 + MyBatis-Plus 分层结构已建立，包含 `entity / mapper / dto / vo / service / controller`。
2. 统一返回结构、全局异常处理、JWT 认证、Spring Security 权限控制已建立。
3. MySQL schema 已覆盖用户、教师资料、分类、课程、章节、收藏、学习记录、直播间、聊天、弹幕、投票、连麦申请、回放、作业、考试、公告、站内信、订单等核心表。
4. 前端 Vue3 + Vite + Vue Router + Tailwind 页面体系已建立，API 统一封装在 `src/lib/api.ts`。
5. 前端已有登录态存储、路由鉴权、暗色玻璃拟态 UI 风格。

### 登录注册与个人基础能力

已完成：

1. 注册、登录、当前用户信息获取。
2. JWT token 存储与请求自动携带。
3. 修改密码后端接口。
4. 头像上传后端接口与 `/uploads/**` 静态访问能力。

待补齐：

1. 前端个人中心不足：资料编辑、头像上传、修改密码入口未闭环。
2. 教师资料自助编辑不足。

### 管理员功能

已完成：

1. 管理总览/平台统计基础版：用户、教师、学生、课程、直播、回放、作业、考试等数量统计。
2. 用户管理基础版：用户分页、角色/状态/关键词筛选、详情、启用/禁用、教师资料编辑。
3. 课程审核：管理员课程列表、状态筛选、审核通过/下架。
4. 分类管理后端：分类列表、新增、编辑、删除接口。
5. 直播管理后端：直播间列表、详情、强制关闭接口。
6. 公告管理：公告列表、新增、编辑、发布、下线、删除，发布时可同步站内信。
7. 站内消息：消息列表、未读数、单条已读、全部已读。

待补齐：

1. 新增教师账号、删除教师、重置密码、子管理员新增与角色分配。
2. 分类管理前端页面。
3. 直播管理前端页面与直播历史查看。
4. 弹幕记录筛选、违规删除、关键词屏蔽。
5. 今日直播场次、平台活跃人数、统计图表增强。

### 教师功能

已完成：

1. 课程管理：课程列表、创建课程、提交审核、章节新增。
2. 直播管理基础版：创建直播间、开始直播、结束直播、查看自己的直播间。
3. 投票发起基础版：教师创建投票并通过 WebSocket 推送。
4. 回放管理：回放列表、新建/编辑/删除回放、上传回放视频、公开视频流播放。
5. 作业管理：发布作业、编辑/删除作业、查看提交列表、批改评分和评语。
6. 考试管理：创建考试、发布/关闭考试、题目增删改、查看提交、批改主观题。
7. 教师统计：教师概览与课程维度统计。

待补齐：

1. 课程章节视频上传到 `/uploads/video/` 及学生端流式播放闭环。
2. 教师个人资料自助编辑、头像上传、修改密码前端入口。
3. Agora Web SDK 真直播：摄像头推流、屏幕共享、麦克风开关。
4. 连麦申请教师端列表、同意/拒绝、音视频连麦。
5. 投票结果统计展示。
6. 直播结束后自动生成回放记录链路。

### 学生功能

已完成：

1. 注册、登录、退出基础能力。
2. 首页基础版：推荐课程、正在直播、近期直播预告。
3. 课程中心：课程列表、分类筛选、关键词搜索、课程详情、章节目录、收藏/取消收藏、学习记录保存接口。
4. 直播中心基础版：直播间列表、详情、WebSocket 在线人数、聊天、弹幕发送、申请连麦。
5. 回放中心：回放列表、回放详情、视频播放。
6. 我的作业：作业列表、详情、提交文本/附件、查看批改结果。
7. 考试：考试列表、答题提交、结果查看。
8. 学生统计：学习课程数、收藏数、作业提交数、考试参与数、平均分、最近学习记录。

待补齐：

1. 个人资料编辑、头像上传、修改密码前端入口。
2. 课程章节视频上传与流式播放闭环。
3. 我的学习页面：记录列表、继续学习、学习进度。
4. 我的收藏页面：收藏课程分页、取消收藏。
5. 直播投票参与面板和结果展示。
6. Agora Web SDK 拉流观看、学生端连麦音视频接入。
